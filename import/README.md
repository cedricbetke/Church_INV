# Import-Ablage

Lege hier die Exporte aus Microsoft Teams ab, die in die Datenbank importiert werden sollen.

Empfohlene Dateien:

- `teams-export.csv`
- optional zusätzlich `teams-export-schema.csv`

Hinweise:

- Die Originaldateien sollten unverändert bleiben.
- Dieser Ordner ist nur als Ablage für den Import-Vorprozess gedacht.
- Der eigentliche Import läuft später über ein separates Script.
